## 一、项目概述

### 1.1 项目定位
面向房屋租赁行业的 AI 智能营销系统。用户上传房源信息（图片/视频/文字），AI 自动生成小红书和公众号风格文案，一键发布到小红书，自动创建公众号草稿，并通过评论区智能引流到私域微信。

### 1.2 目标用户
- 房产中介个人/团队
- 二房东 / 长租公寓运营商
- 租赁行业内容创作者

### 1.3 核心功能
- **AI 智能文案生成**：输入房源信息，DeepSeek 大模型 3 秒生成小红书/公众号风格文案
- **多平台一键发布**：小红书图文笔记全自动发布 + 公众号草稿自动创建
- **智能客服引流**：自动搜索竞品帖子评论区，发送引流话术引导加微信
- **文案自学习**：记录用户修改偏好，逐步优化生成风格
- **内容安全**：敏感词过滤 + 发布频率控制，避免限流封号

## 四、推荐技术架构

### 4.1 架构总览
基于测试结果，推荐以下已验证可行的轻量架构：

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 后端框架 | Python FastAPI | 异步高性能，AI 生态最佳 |
| 前端框架 | React + Vant UI | 移动端 Web App，免安装 |
| AI 大模型 | DeepSeek (deepseek-chat) | OpenAI 兼容，¥0.002/次 |
| 小红书 | xiaohongshu-mcp | 浏览器自动化，已实测发布+评论 |
| 公众号 | 微信官方 API | 自动创建草稿，手动发布 |
| 数据库 | SQLite (起步) | 零运维，月增<1MB，随时可切 MySQL |
| 部署 | 腾讯云轻量服务器 | 2C4G Docker Compose 单机，¥70-150/月 |

### 4.2 核心链路
1. DeepSeek AI 文案 → 小红书 MCP 发布 + 评论引流
2. DeepSeek AI 文案 → 公众号 API 创建草稿
3. 搜索竞品帖子 → 评论区发引流话术 → 引导加微信

### 4.3 存储策略（轻量版）
- 图片/视频上传后即用即清，不长期存储
- SQLite 只存文案文本 + 发布记录 + 风格偏好
- /tmp 临时中转，定时清理残留文件
- 月度数据增长 < 1MB
